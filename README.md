# Dcode_The-easy-way-of-coding
Dcode, a new coding language based on simplicity, no complex syntaxes, 
easy to code, easy to modify
make coding accesible and fun, not boring and complex

Featuress
- colon based syntax
- easy learning curve
- modular libraries (make your own)
- plugin system (create your custom instructions)

installing
download Docde folder on this rrepository, decompress and Run DcodeIDE.EXE adn keep Dcode.pck on the same folder than the.exe
