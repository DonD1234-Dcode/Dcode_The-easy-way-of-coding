# Dcode_The-easy-way-of-coding
Dcode, a new coding language based on simplicity, no complex syntaxes, 
easy to code, easy to modify
make coding accesible and fun, not boring and complex

Featuress
- colon based syntax
- easy learning curve
- modular libraries (make your own)
- plugin system (create your custom instructions)

Changelog
- check CHANGELOG.md for updates!

 Installation
 - check instalation for files

 Usage

- Launch `DcodeIDE.exe` to start the IDE.  
- Keep `DcodeIDE.exe` and `DcodeIDE.pck` together in the same directory.  
- If antivirus flags the file, it is a false positive due to compression.

 Development

- Built with **Godot Engine** (Windows Desktop export).  
- Project stripped and optimized for size (<25 MB executable).  
- Assets compressed with 7‑Zip for GitHub distribution.
